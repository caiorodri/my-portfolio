Bem Vindo

Olá, sou Caio dos Santos Rodrigues e esse é meu portifólio. Sinta-se livre para explorar e conhecer um pouco sobre mim.

Estou cursando Ciências da Computação na UNIP (Universidade Paulista). Não tenho experiência na área, é o que estou buscando neste momento, tenho conhecimento em algumas tecnologias como Python, Java, Angular, HTML, CSS, JavaScript, MySQL, linguagem SQL e Excel. Estou sempre buscando aprender mais e estar atualizado. Atualmente estou estudando Java

Principais Tecnologias

PYTHON - Comecei a estudar python no começo de 2022, desde então venho praticando todos os dias. Fiz alguns projetos pessoais com ele, como Jogo da Forca, Conversor de Moeda, um jogo RPG de texto, e o que eu mais gostei de desenvolver que foi uma Calculadora de Emissão de Carbono para uma APS (Atividade Pratica Supervisionada) da faculdade. Também foi a linguagem escolhida para aprendizado na faculdade em que estou cursando, então consegui rever os conceitos.

HTML e CSS - Comecei a estudar html e css em 2021, fiz alguns cursos e pratiquei bastante. Sou melhor na pratica do que na teoria, comecei a estudar antes que o Python, mas não tenho tanto dominio quanto. Fiz esse portifólio praticamente todo em HTML e CSS. Aproveitei essa ideia para dar uma revisada na linguagem

JAVA - Comecei a estudar java no fim de 2022, mas já vinha estudando o conceito de orientação a objeto antes. Como aprendi muito de Python, estou tendo menos dificuldade em aprender java. Estou tendo aula de java na faculdade atualmente e estou começando alguns projetos pessoais também
